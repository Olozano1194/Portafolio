# Portafolio Personal - Oscar Lozano

Portafolio profesional desarrollado con React, Vite y Tailwind CSS. Un sitio web moderno y responsivo que展示 mis proyectos, habilidades y experiencia como Desarrollador Web.

## 🚀 Características

- **Diseño Responsivo**: Adaptado para dispositivos móviles, tablets y escritorio
- **Modo Claro/Oscuro**: Toggle para cambiar entre temas
- **Internacionalización (i18n)**: Soporte completo para Español e Inglés
- **Navegación Suave**: Scroll fluido entre secciones
- **Proyectos Interactivos**: Sección de demos y videos de proyectos
- **Formulario de Contacto**: Integración con EmailJS

## 🛠️ Tech Stack

- **Frontend**: React 18 + Vite
- **Estilos**: Tailwind CSS v4
- **Idiomas**: react-i18next
- **Formularios**: React Hook Form
- **Icons**: React Icons + Heroicons
- **UI Components**: Headless UI

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/Olozano1194/Portafolio.git

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Build para producción
npm run build
```

## 🌐 Estructura del Proyecto

```
src/
├── assets/           # Imágenes y recursos estáticos
├── components/       # Componentes React
│   ├── layout/       # Header, Footer
│   ├── sections/    # Secciones del portafolio
│   └── language/    # Switcher de idiomas
├── data/             # Datos de proyectos
├── hooks/           # Custom hooks
├── locales/         # Traducciones (ES/EN)
├── pages/           # Páginas
└── utils/           # Funciones utilitarias
```
